# PHPStack

## What is PHPStack,
A simple and tiny docker-compose to run Apache2, NGINX, PHP-FPM and MariaDB+PHPMyadmin



## How to run,
### Run in normal mode
- Clone the repo and run 'docker-compose up' in terminal
### Run in daemon mode
- Clone the repo and run 'docker-compose up -d' in terminal
